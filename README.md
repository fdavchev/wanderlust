# 🌍 Wanderlust — Travel Blog Platform
 
A full-stack travel blog web application built as a university project. Inspired by Instagram's social feed, Wanderlust lets users share travel experiences, discover destinations, and connect through posts pinned to real-world locations on an interactive map.
 
---
 
## Features
 
- **Authentication** — Register and log in with a personal account
- **Posts** — Create posts with a title, description, photo, category, and map location
- **Social interactions** — Like, comment, and bookmark posts
- **Interactive map** — View all post locations pinned on a Leaflet.js map
- **Search** — Filter posts by location, category, or username
- **Profile** — Edit your profile info and view your own posts
- **Bookmarks** — Save posts to revisit later
---
 
## Tech Stack
 
| Layer | Technology |
|-------|------------|
| Backend | PHP |
| Frontend | HTML, CSS, JavaScript |
| Database | MySQL |
| Maps | Leaflet.js + OpenStreetMap |
| Server | Apache (XAMPP / WAMP) |
 
---

 ## Academic Context
 
Built as a university project for the **Web Programming** course at **Goce Delcev University, Shtip** (2023/2024). The goal was to demonstrate full-stack web development skills including user authentication, CRUD operations, file uploads, and third-party API/library integration.
